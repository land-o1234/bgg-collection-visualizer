BGG Collection Visualizer

This is a demonstration of the BoardGameGeek Collection Visualizer.

Features implemented:

  • ✅ BGG XML API client with retry logic
  • ✅ Game similarity computation (Jaccard + Cosine)
  • ✅ Recommendation system via search candidates
  • ✅ JSON data generation (nodes, edges, recommendations)
  • ✅ GitHub Actions workflow for automation
  • ✅ Static web app with interactive graph

Note: The interactive graph requires Cytoscape.js which is blocked in this environment. The full functionality would work when deployed with the library available.
